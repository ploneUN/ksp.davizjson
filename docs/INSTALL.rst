ksp.davizjson Installation
--------------------------

To install ksp.davizjson using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ksp.davizjson`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ksp.davizjson

* Re-run buildout, e.g. with:

    $ ./bin/buildout
